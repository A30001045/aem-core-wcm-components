List

List can be used to display a list of pages. They can be defined either dynamically - by search query, tags or from a parent page - or as a static list of items.

Examples

Items with Link, Description and Date

The rendering of the list items can be configured to optionally display a link to the page, page description and last modified date.

  • Carousel 2019-01-22 Cycle through panels of related content
  • Content Fragment 2019-01-16 Add a Content Fragment asset to a page
  • Image 2019-01-22 Display a single image asset
  • List 2019-01-16 Display a list of pages
  • Separator 2019-01-09 Display a horizontal section divider
  • Social Sharing 2019-01-09 Add social sharing links to a page
  • Tabs 2019-01-22 Organise content into switchable panels
  • Teaser 2019-01-09 Group an image, title, description and link them
  • Text 2019-01-22 Display a rich text paragraph on a page
  • Title 2019-01-16 Create headings for a page or page sections
